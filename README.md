## ğŸš€ Getting Started

### Prerequisites

* Node.js and npm
* Git

### Installation

1. **Clone the repository:**
	```bash
	git clone <your-repo-url>
	```

2. **Navigate to the project:**
	```bash
	cd Final-Frontend
	```

3. **Install dependencies:**
	```bash
	npm install
	```

### Running the App

* Start the application:
  ```bash
  npm run dev
  ```

* Access via your browser: http://localhost:5173
